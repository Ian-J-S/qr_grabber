# QR Grabber

I hate having to get my phone out to scan a QR code unless I have to. This is a simple utility that decodes QR codes from either your system clipboard or an image file.

- From clipboard: Attempts to decode a QR code from the most recently copied image. If your clipboard doesn't currently contain an image, or the image isn't the latest clipboard data, decoding will fail.
- From file: Opens an image file and attempts to decode any QR code it contains.
